## Intro

- PyTorch를 사용하여 ResNet 스타일의 CNN 구현

### CIFAR-10 데이터셋

- 10개의 클래스로 구성된 작은 컬러 이미지 데이터셋
- 총 60,000장 (학습 50,000 + 테스트 10,000)
- class: [ airplane, automobile, bird, cat, deer, dog, frog, hourse, ship, truck ]
